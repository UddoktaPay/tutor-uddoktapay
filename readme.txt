Integrate UddoktaPay payment gateway with Tutor LMS for accepting payments in Bangladesh.

== Description ==

The UddoktaPay Gateway add-on for Tutor LMS allows you to accept payments through various Bangladeshi payment methods including bKash, Nagad, Rocket, and more.

= Features =
* Accept payments through multiple Bangladeshi payment methods
* Seamless integration with Tutor LMS
* Automatic payment verification
* Customizable payment button text
* Easy to configure

= Requirements =
* WordPress 6.2 or higher
* PHP 7.4 or higher
* Tutor LMS plugin
* UddoktaPay merchant account

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/tutor-uddoktapay`, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Tutor LMS → Settings → Payment Methods
4. Enter your UddoktaPay API credentials
5. Save the settings

== Frequently Asked Questions ==

= Do I need a UddoktaPay account? =

Yes, you need a UddoktaPay merchant account to use this plugin. Visit [UddoktaPay](https://uddoktapay.com) to create an account.

= Which payment methods are supported? =

The plugin supports all payment methods available through UddoktaPay, including:
* bKash
* Nagad
* Rocket
* Other Bangladeshi payment methods

= Is SSL required? =

Yes, SSL is required for accepting payments securely through UddoktaPay.
